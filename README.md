# Artefatto:
	In ambito software, un artefatto è qualsiasi prodotto concreto generato, utilizzato o modificato durante il ciclo di vita dello sviluppo del software.
	In altre parole: è qualcosa di tangibile, non un’idea astratta.
	Esempio:Codice sorgente (file .java, .py, .js, ecc.)
			Documentazione
			requisiti funzionali
			specifiche tecniche
			manuali utente
			Diagrammi (UML, architettura, flussi)
			File di configurazione
